# projeto-agenda
Projeto final entregue para a disciplina de Desenvolvimento Web.

## Instalação
Requer banco de dados MySQL ou compatível com URI de conexão na variável `DB_URL` (ou verifique código para variáveis separadas).

Definir `APP_SESSION_SECRET` como string criptograficamente aleatória com ao menos 32 bytes de entropia.

Definir `APP_SESSION_NAME` com nome da aplicação.

Para rodar sessão local de desenvolvimento:
```
npm install
npm run dev
```
